# crypto-market-behavior-analysis
Exploratory data analysis of top cryptocurrencies focusing on price behavior, volatility, market sentiment, and drawdown patterns using historical OHLCV data.
Project Overview

This project analyzes historical price data of the top 50 cryptocurrencies to understand:

* Market behavior (bull vs bear days)

* Price volatility and drawdowns

* Long-term coverage and activity duration

* Risk characteristics across different assets

The goal is to extract actionable insights from raw market data using Python-based exploratory data analysis (EDA).
